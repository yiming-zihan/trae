## 1. Product Overview
家庭菜单管理系统 - 帮助情侣/夫妻轻松管理日常饮食，实现点菜、食材管理和微信消息通知一体化。

- **核心目标**: 简化家庭饮食决策流程，让女方方便点菜，男方及时接收通知，高效管理食材库存
- **用户价值**: 减少沟通成本，避免食材浪费，提升生活品质

## 2. Core Features

### 2.1 User Roles
| Role | Core Permissions |
|------|------------------|
| 女方（点菜方） | 浏览菜品库、选择菜品、提交点菜请求 |
| 男方（接收方） | 接收点菜通知、查看采购清单、管理食材库存 |

### 2.2 Feature Module
1. **菜品管理**: 菜品录入、食材自动生成、菜品分类
2. **点菜系统**: 女方点菜、提交菜单、确认订单
3. **微信通知**: 点菜消息推送、状态更新提醒
4. **食材清单**: 库存管理、采购清单生成、用量计算
5. **菜单历史**: 查看历史记录、重复点菜功能

### 2.3 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 首页 | 菜品浏览 | 分类展示菜品，支持搜索筛选 |
| 点菜页 | 点菜功能 | 选择菜品，设置制作日期 |
| 清单页 | 食材管理 | 库存食材、采购清单 |
| 订单页 | 订单管理 | 历史订单、当前订单状态 |
| 设置页 | 系统设置 | 微信绑定、角色配置 |

## 3. Core Process
```mermaid
flowchart LR
    A[女方浏览菜品] --> B[选择菜品]
    B --> C[设置制作日期]
    C --> D[提交点菜]
    D --> E[微信通知男方]
    E --> F[男方查看订单]
    F --> G[生成采购清单]
    G --> H[采购食材]
    H --> I[更新库存]
```

## 4. User Interface Design
### 4.1 Design Style
- **主色调**: 温暖橙色 (#FF6B35)，传达温馨家庭氛围
- **辅助色**: 清新绿色 (#2EC4B6)，代表新鲜食材
- **按钮风格**: 圆润圆角，渐变色，hover动效
- **字体**: 思源黑体，现代简洁
- **布局**: 卡片式设计，清晰层次

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 首页 | 菜品卡片 | 图片、名称、食材标签、点击选择 |
| 点菜页 | 点菜列表 | 已选菜品、制作日期选择器、提交按钮 |
| 清单页 | 食材列表 | 库存状态标识、采购数量、一键购买 |
| 订单页 | 订单卡片 | 菜品列表、状态标签、时间线 |

### 4.3 Responsiveness
- 移动端优先设计
- 支持触屏操作
- 响应式布局适配各种屏幕尺寸
